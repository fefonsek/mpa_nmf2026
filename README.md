# mpa_nmf2026

This repository was created to store activities and exercises developed in the Financial Mathematics Leveling course, taken in the Professional Master's in Management (MPA) during the first semester of 2026.

## About this repository

This repository contains solutions to financial mathematics exercises implemented using R.  
The main purpose is to organize and document the activities completed throughout the course, while also serving as a reference for anyone interested in learning more about fundamental concepts in financial mathematics.

Some of the topics covered include:
- Simple and compound interest  
- Equivalent interest rates  
- Nominal and effective rates  
- Different capitalization periods  
- Equivalence of capital  
- Interest rate conversions  

## Tools

All exercises are solved using the R programming language, mainly to:
- perform financial calculations
- document the mathematical reasoning
- ensure reproducibility of results

## Purpose

In addition to fulfilling the requirements of the course, this repository also works as a small collection of solved financial mathematics exercises.  
If you are studying the subject or want to see examples of how financial problems can be solved using R, you may find some useful material here.

## Repository structure

The files are organized according to the activities proposed throughout the course. In general, each script includes:
- the problem statement (or a reference to the exercise)
- the reasoning behind the solution
- the R code used to obtain the result

## Note

The scripts were written for educational purposes and may include explanatory comments (in portuguese) to make the calculation steps easier to understand.
